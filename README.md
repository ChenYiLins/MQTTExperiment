# MQTT Experiment

This is a classroom lab project designed to easily upload data collected by the ESP32 to the cloud via the MQTT protocol.

## Overview

- Connect to server via the MQTT/MQTTS protocol (it use MQTTS now)
- Read the temperature data from DH18B20
- Read the humidity data from network servers (broad miss DH11 sensor)
- Storing MQTT connection parameters through NVS
- Python server established by Flask library realizes the function similar to that of upper computer of single chip microcomputer
